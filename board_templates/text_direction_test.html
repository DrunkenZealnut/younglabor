✅ 수정 후

  • • 모든 텍스트가 가로 방향으로 정상 출력
  • • 한글 폰트 우선순위 설정 완료
  • • CSS `!important`로 강제 적용

🔧 적용된 CSS 수정사항

/* 텍스트 세로 출력 방지 */
.board-surface,
.board-surface *,
table, th, td {
    writing-mode: horizontal-tb !important;
    text-orientation: mixed !important;
    direction: ltr !important;
}

/* 한글 폰트 우선순위 */
.board-surface {
    font-family: 'Noto Sans KR', 'Malgun Gothic', '맑은 고딕' !important;
}
